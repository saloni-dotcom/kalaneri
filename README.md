# Active Management Suite
A dashboard analytics and management tool built using Claude Code, 
connected to Firebase for real-time data storage and retrieval.
## Features
- Dashboard with live analytics
- Data management interface
- Firebase real-time database integration
- Built and deployed using AI-assisted development (Claude Code)
## Tech Stack
- Claude Code (primary development tool)
- Firebase (backend/database)
- HTML / JavaScript
## About
Built as a personal project to explore AI-assisted development 
and demonstrate practical implementation of AI tools in building 
real-world applications.
**Built:** May 2026
